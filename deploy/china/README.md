# GRIPLAB 中国大陆部署说明

此版本可以部署到腾讯云或阿里云的中国内地服务器。核心鼠标数据已经包含在站点中，浏览器只访问自己的域名。

## 上线前准备

1. 注册并完成实名认证的域名。
2. 购买中国内地云服务器或轻量应用服务器。
3. 在服务器所属云厂商提交 ICP 备案；备案完成前不要把域名解析到中国内地服务器并开放网站。
4. 推荐创建同厂商的 COS/OSS 对象存储，用于保存已经取得使用许可的产品图片。

## 环境配置

- `NEXT_PUBLIC_SITE_URL`：正式站点地址，例如 `https://mouse.example.com`。
- `MOUSE_IMAGE_ORIGIN`：可选，国内图片存储目录，例如 `https://static.example.com/mouse-images/`。目录中的文件名需要与 `public/mice-database.json` 的 `image` 字段一致。没有设置时，服务器会继续从现有公开图片源代理图片，因此不属于完全本地化。

## 容器部署

使用仓库根目录的 `Dockerfile.china` 构建并运行。对外仅开放 HTTPS，由云负载均衡或 Nginx 转发到容器的 3000 端口。

上线后应在网页底部展示 ICP 备案号；如适用，再按所在地要求办理公安联网备案。
